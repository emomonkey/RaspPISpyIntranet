RaspPISpyIntranet

Flask Python Application that integrates with a Webcam or Raspberry PI Camera. Camerashots can be trigger via a sensor or in autoshot

Sensor Mode

The application can be triggered by a PIR sensor that is connected to GPIO 17. If RaspSpyIntranet is in alarm mode a camerashot will be taken. The picture will also be emailed

Random Mode

Photos can also be taken randomly.The frequency and times of these random shots can be figured. This is so random fun shots can be taken to keep a record. You can set an end time of when the random photos are taken. They can also be emailed out or deleted.

Configuring

All the settings can be configured via the Setup screen. They are stored locally in a sqlite database. This screen is used to setup items such as email or the type of camera connected
